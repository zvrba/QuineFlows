      Quine.FileWatcher Namespace

      Contains types for defining hierarchies of "watch folders".

      Classes

      ErrorInfoPathEquality Defines equality of ErrorInfo instances as equlity of their Path members. This is a stateless class and cannot be instantiated directly; use Instance.
      IErrorContextDefault Default implementation. The Accept(ErrorInfo) method just returns the incoming exception. This class is stateless and cannot be instantiated directly; use Instance instead.
      SimpleDirectoryMonitor Recursively watches a directory for new files by polling so that files being written to are delivered only after their size has not changed since the last call to Update. This class is NOT thread-safe.
      TraversalVisitor Provides methods for traversing the filesystem.
      WatchConflictException Thrown when conflicting child is attempted added to a watch node. The path properties DO NOT include the root node's path.
      WatchNode A node in the "watch tree" which watches directory hierarchy for stable files.
      WatchResultEntryPathComparer Implements comparison and equality over Path. The comparison used is case-sensitive ordinal. This class is stateless and cannot be instantiated directly; use Instance instead.

      Structures

      ErrorInfo Describes errors encountered during file/directory operations.
      WatchResult Collection of watch results.
      WatchResultEntry Single entry reported by walking the watch tree.

      Interfaces

      IErrorContext Provides an error-handling context.

      Delegates

      TraversalFilter Delegate type used by TraversalVisitor.
      WatchParameterParser Delegate type for parsing parameter nodes into values. Implementations MUST NOT throw.

      Enumerations

      ErrorCode Classifies the error instance reported by ErrorInfo.
      TraversalFilterInstruction Used as argument to TraversalFilter.
