      IQdbConnection Interface

      Provides access to transactions and table operations. Default method implementations are targeted towards @-style parameters which are supported by most DB providers.

      Properties

      Database Owning connection factory.
      DbConnection The underlying DbConnection.
      Transaction Non-null if a transaction is active. Only one transaction can be active at a time and nested transactions are not supported.

      Methods

      BeginTransactionAsync Begins a transaction if possible.
      CreateCommand Creates a command tied to this connection and transaction if any is active at the time of invocation.
      CreateDeleteTEntity Creates a reusable command for deleting an entity.
      CreateFindTEntity Creates a reusable command for finding an entity by key.
      CreateInsertTEntity Creates a reusable command for inserting an entity.
      CreateUpdateTEntity Creates a reusable command for updating an entity. This method cannot be used to modify key fields.
      DeleteAsyncTEntity Attempts to delete entity by its primary key.
      FindAsyncTEntity Finds an entity by its primary key.
      GetSequenceNumber Atomically retrieves the next available number from a named sequence. This method must be invoked within an active transaction.
      InsertAsyncTEntity Creates a new entity. An exception is thrown if the key already exists. Specifying a non-default value for accessor allows use of projections for TEntity.
      UpdateAsyncTEntity Updates an existing entity. This method cannot be used to modify key fields. Specifying a non-default value for accessor allows use of projections for TEntity.
