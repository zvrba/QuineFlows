      IFileStreamOpenStrategy Interface

      Provides methods for creating unbuffered file streams from paths directly accessible to OS.

      Definition

      Namespace: Quine.FileTransfer
      Assembly: Quine.FileTransfer (in Quine.FileTransfer.dll) Version: 1.0.0+2d4678028124cbb40154ed2e4b22472910ab3869
      C#
      public interface IFileStreamOpenStrategy

      Properties

      Default Default instance: supports Windows and OSX.

      Methods

      OpenRead Opens a file for unbuffered reading. Must throw exception on failure.
      OpenWrite Opens and truncates a file for unbuffered writing and reading, overwriting previously existing file. Must throw exception on failure. (Read mode is necessary for optional hash verification.)
      SetDefault Allows replacing Default. This is intended to support other operating systems.
