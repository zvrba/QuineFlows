      WatchNode Methods

      Methods

      Clone Replaces root path and error context in the tree rooted at root and returns a new tree.
      CreateWatchConflictException Creates an instance of WatchConflictException with conflict details filled in.
      MakeChild Makes a child node or returns an existing one matching pattern.
      MakeRoot Creates the root of the watch tree without a root path or error context. The tree must be cloned (Clone(WatchNode, PathComponents, IErrorContext)) to be actually usable for walking the file-system (Walk(WatchNode)).
      Walk Walks the watch tree and reports results.
