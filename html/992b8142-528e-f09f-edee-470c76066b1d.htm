      IFileStreamOpenStrategy Methods

      Methods

      OpenRead Opens a file for unbuffered reading. Must throw exception on failure.
      OpenWrite Opens and truncates a file for unbuffered writing and reading, overwriting previously existing file. Must throw exception on failure. (Read mode is necessary for optional hash verification.)
      SetDefault Allows replacing Default. This is intended to support other operating systems.
