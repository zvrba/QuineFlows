      Samples

      The libraries are delivered as a Visual Studio solution. Quine.Samples is a runnable command-line application that demonstrates how to get quickly started with using the libraries. The samples demonstrate only key features, so use the reference documentation to discover other features. They were also written in a hurry, so please excuse the messy code.

      List of samples

      The sample project includes a stress test for Quine.FileTransfer library. For this, it defines a synthetic data producer and a consumer that generate easy-to-validate data. These can be used as another example of creating new (parallell) producers and consumers. They are able use AlignedBuffer class due to InternalsVisibleTo declaration in Quine.FileTransfer. The class is not exposed publicly because it is "dangerous" -- careless use would open for use-after-free bugs which can trigger undefined behavior (data corruption, crashes) and cause security problems.

        Important

      Study the code and comments in Quine.Samples\Fixture.cs because they explain some subtleties around reported exceptions.
