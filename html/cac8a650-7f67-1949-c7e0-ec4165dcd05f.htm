      INormalStringTraits Interface

      Provides methods that determine rules for string normalization, comparisons, etc. used by NormalStringTStringTraits. This interface should be implemented by a stateless struct. All implementations must add themselves to KnownTypes in their static ctor so that NormalStringTStringTraits can be serialized and deserialized without error.
