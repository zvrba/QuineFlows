      Quine.Schemas Namespace

      Contains support classes for the serialization infrastructure. You probably won't need to use those.

      Classes

      DataContractSchemaExporter 
      DCSerializer Simple methods for (de)serializing types with DataContractSerializer to XML.
      SchemaLoader Helper class for loading schemas. Schemas must be loaded prior to using methods in DCSerializer because all types must be registered with DataContractSerializer upfront.
      XmlSerializerSchemaExporter Utility class to load all assemblies from the application's directory and export them to XSD. Classes in Quine.Schemas.Externals will NOT be exported. Schemas are written to files named Schema{0:D2}.xsd.
      XSerializer Simple methods for (de)serializing types with XmlSerializer.
