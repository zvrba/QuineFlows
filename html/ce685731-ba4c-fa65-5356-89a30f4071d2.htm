      Remarks

      This is a somewhat unfortunate API because various parameter combinations are used to create all 3 kinds of watch nodes.
      Constant node Created when pattern is a string NOT enclosed in $(). In this case, parser and value must both be null.
      Parameter node Created when pattern is a string enclosed in $() (a "parameter"), for example $(DayNumber). In this case, parser MUST be provided, and value MUST be null. The dictionary in Parameters will have the parameter name (without enclosing $()) as the key and the object returned by parser as value.
      Leaf node Created when pattern is a parameter and value is NOT null. As above, the dictionary in Parameters will have the parameter name as the key and provided value as the value.
