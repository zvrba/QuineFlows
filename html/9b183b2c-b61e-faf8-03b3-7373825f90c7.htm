      ErrorContext Error notifier. If null, the exception will just be rethrown.
      Filter Traversal filter, used to skip particular files and directories. If null, all entries are reported.

      Methods

      GetDirectories Enumerates directories in directory as determined by Filter. Note that this method does NOT return directory itself. The filter must return Recurse flag in order to recurse into subdirectories.
      GetFiles Enumerates files in directory. The enumeration honours Accept flag returned by Filter. Recursion is determined by recurse parameter.

      Events

      PathDiscarded Subscribe to this event to receive notifications about files that the enumeration discarded.
