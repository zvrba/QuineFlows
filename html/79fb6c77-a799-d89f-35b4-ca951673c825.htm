      QdbValueAccessorTEntity Class

      Provides weakly-typed access to an arbitrary (possibly nested) member of TEntity. Equality is based on DbName field since names must be unique in any parameter set or table. The two instances are "equal" if they refer to the same property OR have the same DbName attribute. (A collection of value accessors can contain unique instance of either.)
