      Remarks

      Usage notes: the class implementing IQdbConnection can subscribe to OnDisposed and ensure that output parameters are written back to the parameter block after disposal. Dispose methods automatically clear OnDisposed event, so unsubscription is not necessary.

      Methods

      DisposeReleases all resources used by the QdbReader
      EnumerateResultSetAsyncTEntity Asynchronously enumerates the current result set, one item at a time. On completion, the reader is positioned to the next result set. HasNextResultSet is set to true if the next result set exists, otherwise false.
      ReadResultSetAsyncTEntity Reads all entities from the current result set and adds them to entities. On completion, the reader is positioned to the next result set and HasNextResultSet is set to true if the next result set exists.
